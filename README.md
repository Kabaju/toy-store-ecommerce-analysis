# Toy Store E-Commerce Performance Analysis

## Objective
Analyze revenue, profit, and margin trends across products and time to evaluate overall business performance.

## Tools Used
- PostgreSQL
- SQL
- Power BI
- Excel

## Dataset
Maven Analytics Toy Store E-Commerce Dataset

## Methodology
- Created a PostgreSQL database from raw CSV files
- Wrote SQL queries to calculate revenue, profit, and profit margins
- Aggregated results monthly
- Exported cleaned data for visualization in Power BI
- Built an interactive dashboard with custom tooltips and KPIs

## Key Insights
- Profit margins remained consistently high (≈59%–68%) across all products
- Revenue showed steady month-over-month growth
- Margin stability suggests strong pricing efficiency

## Deliverables
- SQL queries
- Aggregated dataset
- Power BI dashboard snapshot
